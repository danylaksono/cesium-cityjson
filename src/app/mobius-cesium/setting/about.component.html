cesium-viewer

Version 0.2.2

This is a viewer for CityGML and CityJSON based on Ceisum. This project was generated with Angular CLI.

It aims to open and view models with the ability to:

  • View models on a map (if location is provided)
  • Select individual objects or surfaces and view attributes
  • Filter visible objects based on extracted attributes

Github Repository can be found here.

Examples

You may load your own model by clicking on the file button in the top right corner. If not, some examples can be loaded from the selection below.
